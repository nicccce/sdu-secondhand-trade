### 部署

1. 克隆项目代码：

   ```
   git clone https://github.com/nicccce/sdu-secondhand-trade.git
   ```

2. 复制示例配置文件并进行必要修改：

   ```
   cp ./sdu-secondhand-trade-backend/config.yml.example config.yml
   ```

3. 根据实际情况填写 `config.yml` 中的相关信息

4. 在后端目录中启动项目：

   ```
   go build
   ./project-name
   ```
